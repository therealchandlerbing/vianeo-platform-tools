# STEP 1: Application Form Prompt

**Time Required:** 15 minutes
**Optional:** Skip this step if not managing formal program applications

---

## Purpose

Populate standardized application formats using the VIANEO Canvas from Step 0. This step is useful for:
- Managing multiple startups in a cohort or program
- Tracking applications in a database or CRM system
- Standardized reporting requirements
- Program management and milestone tracking

**Skip this step if:**
- Conducting ad-hoc evaluations
- No program management system needed
- Direct assessment without formal application process

---

## Instructions for AI Assistant

Using the VIANEO Canvas completed in Step 0, populate the appropriate standardized application format below.

Choose the format that matches your program:
- **360 SIS Format** - For international programs
- **CNEN Format** - For Brazilian programs
- **Custom Format** - Adapt as needed for your organization
