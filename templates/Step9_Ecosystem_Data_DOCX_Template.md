# Step 8 DOCX: Ecosystem Data Matrix - Formatting Specifications

**Purpose:** Structured reference document showing complete ecosystem across 5 value chain positions with priority targets highlighted

**Primary Audiences:**
- Investment committees (ecosystem validation)
- Strategic planning teams (market entry strategy)
- Board members (stakeholder landscape overview)
- Partnership development (identification of key players)

**Use Cases:**
- Committee evaluations
- Strategic planning sessions
- Investor due diligence
- Team onboarding and alignment

---

## Document Structure

### Title Page

**Main Title:** (20pt, Bold, Color: #1a1a1a)
```
[PROJECT NAME] ECOSYSTEM VALUE NETWORK DATA
```

**Spacing:** 6pt after title

**Metadata Block:** (10pt, Regular, Color: #6c757d)
```
Last Updated: [YYYY-MM-DD]
Analyst: [Your Name]
Project Stage: [Idea / Prototype / Pilot / Early Commercialization]
```

**Styling:**
- Each line 6pt spacing
- Light gray bottom border (1pt, #e9ecef)
- 18pt spacing after block

---

## PROJECT OVERVIEW Section

**Section Header:** (16pt, Bold, Color: #2E5C8A)
```
PROJECT OVERVIEW
```

**Horizontal Line:** (2pt, Color: #e9ecef)

**Spacing:** 12pt after header, 12pt after line

### Content Format

**Field Labels:** (11pt, Bold)
```
Project Name:
Industry/Domain:
Core Solution:
```

**Field Content:** (11pt, Regular)
- Labels bold, content regular
- 6pt spacing after each line
- Character limits noted in parentheses (grayed out in final doc)

## ENABLERS & INFLUENCERS Section

**Section Header:** (14pt, Bold, Color: #42a5f5)
```
ENABLERS & INFLUENCERS
```

**Horizontal Line:** (2pt, Color: #e9ecef)

**Subheading:** (10pt, Italic, Color: #6c757d)
```
Organizations providing infrastructure, funding, regulation, or expertise
```

**Spacing:** 8pt after header, 6pt after subheading, 8pt before table

### Table Format

**Table Style:** Grid Table 4 - Accent 1 (or custom with specifications below)

**Column Widths:**
- Organization: 18%
- Role/Description: 22%
- Requester: 16%
- Acceptability: 12%
- Need Level: 12%
- Notes: 20%

**Header Row:**
- Background: #42a5f5 (light blue)
- Text: White, 10pt, Bold
- Padding: 8pt all sides
- Alignment: Center

**Data Rows:**
- Text: 10pt, Regular, Black
- Padding: 6pt all sides
- Alignment: Left (except Acceptability and Need Level: Center)
- Row spacing: Exactly 14pt

**Priority Target Rows** (Favorable + Critical/Important):
- Background: #e8f5e9 (light green)
- Border: 2pt, #4caf50 (green), all sides

**Favorable Rows** (no urgent needs):
- Background: White
- Border: 1pt, #66bb6a (light green), left side only

**Neutral Rows:**
- Background: #fff8e1 (light yellow/orange)
- Border: 1pt, #ffa726 (orange), left side only

**Unfavorable Rows:**
- Background: #ffebee (light red)
- Border: 1pt, #ef5350 (red), left side only

**Spacing:** 12pt after table

**Horizontal Line:** (2pt, Color: #e9ecef)

**Spacing:** 8pt after header

### Product Name Box

**Box Styling:**
- Background: Linear gradient #667eea to #764ba2 (or solid #667eea)
- Border: None
- Padding: 12pt all sides
- Border radius: 4pt (rounded corners if Word supports)

**Product Name:** (14pt, Bold, White)
```

```

**Tagline:** (11pt, Regular, White/95% opacity)
```
[Tagline or brief value proposition]
```

**Spacing:** 6pt between name and tagline

### Key Features List

**List Style:** Bulleted with custom bullet (✓ if available, or solid circle)

**Text:** (10pt, Regular, White/95% opacity if in box, or Black if outside)

```
✓ [Feature 1]
✓ [Feature 2]
✓ [Feature 3]
✓ [Feature 4]
✓ [Feature 5]
✓ [Feature 6]
```

**Bullet Color:** #4ade80 (bright green) if in colored box

**Spacing:** 3pt between items, 12pt after list

## Tips for DOCX Creation

1. **Start with a template:** Use Word's professional report template or create custom styles
2. **Use Styles:** Define paragraph styles for each section type (saves time on formatting)
3. **Table of Contents:** Add auto-generated TOC at beginning for longer documents
4. **Color consistency:** Use Word's color picker with exact hex codes for brand consistency
5. **Save as template:** Once formatted, save as .dotx template for future use
