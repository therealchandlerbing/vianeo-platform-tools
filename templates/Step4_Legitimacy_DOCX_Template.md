## DOCX Formatting Specifications

When creating the DOCX output:

### Document Setup
- **Page Size:** Letter (8.5" Ã— 11")
- **Margins:** 1" all sides
- **Orientation:** Portrait
- **Font:** Calibri
- **Base Font Size:** 11pt
- **Line Spacing:** 1.15, 6pt spacing after paragraphs

### Title Formatting
- **Document Title:** Bold, 14pt, centered, black (#000000)
- **Project/Meta Info:** Regular, 11pt, centered or left-aligned
- **Spacing After Title:** 12pt

### Section Header Formatting
- **Main Headers (Core Info, Human Resources, etc.):** Bold, 12pt, left-aligned, black
- **Sub-Headers (Strengths, Gaps, etc.):** Bold, 11pt, left-aligned, black
- **Spacing After Headers:** 6pt

### Table Formatting
- **Table Style:** Light Grid Accent 1 (or Grid Table 1 Light)
- **Header Row:** Bold text, light gray background (#F2F2F2)
- **Column Widths:**
  - Core Info: Field 25%, Content 75%
  - Human Resources: Means 50%, Differentiator 15%, Note 35%
  - Physical/Intellectual: Means 45%, Differentiator 15%, Note 40%
  - Financial: Resource 30%, Amount 15%, Differentiator 15%, Note 40%
- **Cell Padding:** 0.05" all sides
- **Borders:** 1pt solid, light gray (#D3D3D3)
- **Alternating Row Colors:** Optional - white and very light gray (#FAFAFA)

### Score Color Coding
Apply color to score values (both overall and component scores):

| Score Range | Category | Text Color |
|------------|----------|------------|
| 4.5-5.0 | Exceptional | Dark green (#0A6522) |
| 3.5-4.4 | Above Average | Green (#28A745) |
| 3.0-3.4 | Adequate | Dark amber (#CC8800) |
| 2.0-2.9 | Below Average | Orange (#DD7700) |
| <2.0 | Poor | Red (#DC3545) |

### Text Formatting
- **Body Text:** Regular, 11pt, Calibri, black (#000000)
- **Bold Elements:**
  - All section headers
  - Table headers
  - Field labels ("Overall Note:", "Content (250 chars max):", etc.)
  - Score labels
- **Italic Elements:** None required
- **Bullet Lists:**
  - Standard round bullets
  - 0.25" indent from left margin
  - 6pt spacing after each item

### Spacing Standards
- **After Title:** 12pt
- **After Section Headers:** 6pt
- **After Paragraphs:** 6pt
- **After Tables:** 12pt
- **Between List Items:** 6pt
- **Line Spacing:** 1.15 throughout

### Page Breaks
- Avoid page breaks mid-table
- Keep "Detailed Scoring" component explanations together
- Insert page break before "Assessment Summary" if document length requires

### Professional Elements
- **Page Numbers:** Bottom center, "Page X of Y" format, 9pt
- **Footer:** Assessment date and assessor name (small, 9pt, gray #666666)
- **Watermark (Optional):** "DRAFT - INTERNAL REVIEW" diagonal, very light gray if needed

## Python Code Examples (Reference)

### Document Initialization

```python
from docx import Document
from docx.shared import Pt, RGBColor, Inches
from docx.enum.text import WD_ALIGN_PARAGRAPH

def create_legitimacy_worksheet(project_name):
    """Initialize document with proper styling"""
    doc = Document()

    # Set default font
    style = doc.styles['Normal']
    font = style.font
    font.name = 'Calibri'
    font.size = Pt(11)

    return doc
```

### Add Title

```python
def add_title(doc, project_name):
    """Add main title"""
    title = doc.add_heading(f'Vianeo Legitimacy Worksheet: {project_name}', level=1)
    title_format = title.runs[0]
    title_format.font.size = Pt(14)
    title_format.font.bold = True
    title_format.font.color.rgb = RGBColor(0, 0, 0)
    title.paragraph_format.space_after = Pt(12)
    title.alignment = WD_ALIGN_PARAGRAPH.CENTER
```

### Add Core Information Table

```python
def add_core_info_table(doc, genesis, problem, field):
    """Create Core Project Information table"""

    # Add section header
    header = doc.add_heading('Core Project Information', level=2)
    header.runs[0].font.size = Pt(12)

    # Create table
    table = doc.add_table(rows=4, cols=2)
    table.style = 'Light Grid Accent 1'

    # Header row
    header_cells = table.rows[0].cells
    header_cells[0].text = 'Field'
    header_cells[1].text = 'Content'

    # Make header row bold
    for cell in header_cells:
        for paragraph in cell.paragraphs:
            for run in paragraph.runs:
                run.font.bold = True

    # Data rows
    table.rows[1].cells[0].text = 'Project Genesis'
    table.rows[1].cells[1].text = genesis

    table.rows[2].cells[0].text = 'Problem to Solve'
    table.rows[2].cells[1].text = problem

    table.rows[3].cells[0].text = 'Field of Application'
    table.rows[3].cells[1].text = field

    # Add spacing after table
    doc.add_paragraph()
```

### Add Human Resources Table

```python
def add_human_resources_table(doc, team_members, overall_note):
    """Create Human Resources table"""

    # Add section header
    header = doc.add_heading('Available Means: Human Resources', level=2)
    header.runs[0].font.size = Pt(12)

    # Create table (1 header + team members)
    num_rows = 1 + len(team_members)
    table = doc.add_table(rows=num_rows, cols=3)
    table.style = 'Light Grid Accent 1'

    # Header row
    header_cells = table.rows[0].cells
    header_cells[0].text = 'Means'
    header_cells[1].text = 'Differentiator?'
    header_cells[2].text = 'Note'

    # Make header row bold
    for cell in header_cells:
        for paragraph in cell.paragraphs:
            for run in paragraph.runs:
                run.font.bold = True

    # Data rows
    for idx, member in enumerate(team_members, start=1):
        table.rows[idx].cells[0].text = member['name']
        table.rows[idx].cells[1].text = member['differentiator']
        table.rows[idx].cells[2].text = member['note']

    # Add overall note
    doc.add_paragraph()
    note_para = doc.add_paragraph()
    note_para.add_run('Overall Note (250 chars): ').bold = True
    note_para.add_run(overall_note)

    doc.add_paragraph()
```

### Add Assessment Summary with Scoring

```python
def add_assessment_summary(doc, score, category, strengths, gaps,
                          detailed_scores, overall_assessment,
                          risk_factors, success_factors):
    """Add comprehensive assessment summary section"""

    # Main header
    header = doc.add_heading('Assessment Summary', level=2)
    header.runs[0].font.size = Pt(12)

    # Score with color coding
    score_para = doc.add_paragraph()
    score_run = score_para.add_run(f'Legitimacy Score: {score}/5 ({category})')
    score_run.bold = True
    score_run.font.size = Pt(12)

    # Apply color based on score
    score_val = float(score)
    if score_val >= 4.5:
        score_run.font.color.rgb = RGBColor(10, 101, 34)  # Dark green
    elif score_val >= 3.5:
        score_run.font.color.rgb = RGBColor(40, 167, 69)  # Green
    elif score_val >= 3.0:
        score_run.font.color.rgb = RGBColor(204, 136, 0)  # Amber
    elif score_val >= 2.0:
        score_run.font.color.rgb = RGBColor(221, 119, 0)  # Orange
    else:
        score_run.font.color.rgb = RGBColor(220, 53, 69)  # Red

    doc.add_paragraph()

    # Strengths
    strengths_header = doc.add_paragraph()
    strengths_header.add_run('Strengths:').bold = True
    for strength in strengths:
        doc.add_paragraph(strength, style='List Bullet')
    doc.add_paragraph()

    # Gaps
    gaps_header = doc.add_paragraph()
    gaps_header.add_run('Gaps:').bold = True
    for gap in gaps:
        doc.add_paragraph(gap, style='List Bullet')
    doc.add_paragraph()

    # Detailed Scoring
    detailed_header = doc.add_paragraph()
    detailed_header.add_run('Detailed Scoring:').bold = True
    doc.add_paragraph()

    for idx, (component, score_val, explanation) in enumerate(detailed_scores, 1):
        score_para = doc.add_paragraph()
        score_run = score_para.add_run(f'{idx}. {component}: {score_val}/5')
        score_run.bold = True

        # Apply color to component score
        score_num = float(score_val)
        if score_num >= 4.5:
            score_run.font.color.rgb = RGBColor(10, 101, 34)
        elif score_num >= 3.5:
            score_run.font.color.rgb = RGBColor(40, 167, 69)
        elif score_num >= 3.0:
            score_run.font.color.rgb = RGBColor(204, 136, 0)
        elif score_num >= 2.0:
            score_run.font.color.rgb = RGBColor(221, 119, 0)
        else:
            score_run.font.color.rgb = RGBColor(220, 53, 69)

        score_para.add_run(' ' + explanation)
        doc.add_paragraph()

    # Overall Assessment
    assessment_header = doc.add_paragraph()
    assessment_header.add_run('Overall Assessment:').bold = True
    doc.add_paragraph(overall_assessment)
    doc.add_paragraph()

    # Key Risk Factors
    risks_header = doc.add_paragraph()
    risks_header.add_run('Key Risk Factors:').bold = True
    for risk in risk_factors:
        doc.add_paragraph(risk, style='List Bullet')
    doc.add_paragraph()

    # Key Success Factors
    success_header = doc.add_paragraph()
    success_header.add_run('Key Success Factors:').bold = True
    for factor in success_factors:
        doc.add_paragraph(factor, style='List Bullet')
```
