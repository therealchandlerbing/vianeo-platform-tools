## The 1-5 Scoring Scale

| Score | Meaning | Evidence Level |
|-------|---------|----------------|
| **5** | Exceptional | 15-20+ external validations, multiple sources |
| **4** | Strong | 10-15 external validations, clear evidence |
| **3** | Adequate | 5-10 external validations (minimum threshold) |
| **2** | Weak | 1-4 validations, mostly assumptions |
| **1** | Insufficient | No external validation, pure assumption |

**Key Rule**: Score what EXISTS, not what's PLANNED

## Quick Tips

1. **Start with Desirability** - Usually has most evidence
2. **Score conservatively** - Better to underestimate and improve
3. **Be specific** - Use numbers, names, dates
4. **External only** - Internal beliefs don't count for Score 3+
5. **One sentence** - Keep justifications concise
6. **Cite sources** - Name the document, interview, or data
7. **Specify needs** - For Score 1, say exactly what's required
8. **Check consistency** - Related questions should align
9. **Identify patterns** - Multiple Score 1s = systemic issue
10. **Focus on action** - Use results to drive improvement
